# Confused AD Experiment

**「この広告は、あなたの"困惑"を読んでいる」**

ユーザーの打鍵リズムやマウス操作からリアルタイムで困惑度を推定し、広告表示を動的に変化させるインタラクティブな実験デモです。

## デモ概要

ユーザーが広告横のテキストエリアに文章を要約入力している間、以下の行動データをリアルタイム計測します。
推定された困惑スコアに応じて、広告のテーマや色調が自動的に変化します。

## 計測指標

| 指標 | 内容 |
|------|------|
| 打鍵間隔（IKI） | キー入力の速度・リズムから入力の流暢性を判定 |
| 停滞時間 | キー入力なしの経過時間 |
| 削除・修正回数 | バックスペース使用の頻度 |
| マウス往復 | X方向の行き来の回数 |
| マウス速度 | 移動速度の平均値 |

困惑スコア（0〜100）は250msごとに更新され、スコア帯に応じて4段階のテーマを切り替えます。

## 技術スタック

- HTML / CSS / JavaScript（Vanilla）
- センサーデータ（キーボード・マウスイベント）のリアルタイム処理
- 重み付きスコア合成アルゴリズム

## 背景・動機

修士研究「マルチモーダルデータを用いた文章要約作業における困惑状態推定」の副産物として制作しました。
研究では視線・打鍵・座圧センサーを用いてPythonで困惑検知モデルを構築していますが、
本プロジェクトはその概念をブラウザ上でJavaScriptのみで実装し、広告UXへの応用可能性を探ったプロトタイプです。

> 研究本体：scikit-learn / TensorFlow / スタッキングアンサンブルで分類精度0.81達成

## 実行方法

`index.html` をブラウザで開くだけで動作します（サーバー不要）。

```bash
open index.html
```
